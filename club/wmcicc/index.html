WMCI Computing Competition [Annual Programming Competition]

The WCMI Computing Competition is an annual programming competition run at Walter Murray. The competition follows the same format as the University of Waterlooâ€™s Computing Competition (similar to the Junior difficulty level), which means there are 5 questions, and participants are given 3 hours to solve as many of the questions as they can.

Programming Languages

The programming languages that are currently allowed in the WMCI Computing Competition are:

  • Python (3.x)
  • C or C++
  • Java

Allowed Reference Material

Any written material is allowed (textbooks, reference sheets, etc). Online references are restricted to the following official reference pages:

  • Python Docs
  • C/C++ Reference
  • Java Docs

Use of any other websites results in disqualification from the competition.
