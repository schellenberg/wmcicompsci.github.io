WMCI Comp Sci Honour Code

The purpose of computer science classes at Walter Murray Collegiate is to teach you how to think. In order for you (and other students) to achieve this goal, there are a few basic groundrules you need to commit to:

  • Cite any help that you have received (from other students, forums, etc.) on your assignments
  • Do not share actual program code with others. You are ENCOURAGED to discuss your approaches to solving problems, but you should not share actual code.
  • Be ready to explain any parts of the code you submit. If you don’t understand what something does, it doesn’t belong in your assignment.
